# Settings > Secrets and variables > Actions > New repository secret

# ========================================
# ZORUNLU SECRETS
# ========================================

Name: OPENAI_API_KEY
Value: sk-proj-your-actual-openai-api-key-here
Description: OpenAI API key (https://platform.openai.com/api-keys)

# ========================================
# E-POSTA SECRETS (Opsiyonel)
# ========================================

Name: LEAD_MAIL_TO
Value: info@kayacuneyt.com
Description: Lead bildirimlerinin gönderileceği e-posta

Name: SMTP_HOST
Value: smtp.gmail.com
Description: SMTP sunucu adresi (Gmail: smtp.gmail.com, Outlook: smtp-mail.outlook.com)

Name: SMTP_PORT
Value: 587
Description: SMTP port (587 = TLS, 465 = SSL)

Name: SMTP_USER
Value: your-email@gmail.com
Description: SMTP kullanıcı adı (genelde e-posta adresiniz)

Name: SMTP_PASS
Value: your-gmail-app-password
Description: SMTP şifre (Gmail için App Password - normal şifre DEĞİL!)
Note: Gmail App Password almak için: https://myaccount.google.com/apppasswords

# ========================================
# WEBHOOK SECRET (Opsiyonel)
# ========================================

Name: LEAD_WEBHOOK_URL
Value: https://hooks.zapier.com/hooks/catch/your-webhook-url
Description: Lead'leri göndermek için webhook URL (Zapier, Make, Google Sheets vb.)

# NOTLAR

1. Secrets ekledikten sonra değişiklikleri görmek için yeniden deploy gerekebilir
